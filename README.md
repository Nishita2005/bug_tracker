# 🛡️ Bug Tracker System
A comprehensive, role-based Bug Tracking System built with **PHP** and **MySQL**. This application allows teams to manage software defects through a structured workflow involving Admins, Testers, and Developers.

## 🚀 Features
- **Admin Dashboard**: Full control over users and projects. Assign bugs to specific developers with one click.
- **Tester Dashboard**: Report new issues, track the status of reported bugs, and verify/close fixed issues.
- **Developer Dashboard**: View a personalized list of assigned bugs and update their status (Open, In Progress, Fixed).
- **Audit Log (Activity Trail)**: Transparent history of every status change, tracking who did what and when.
- **Advanced Search**: Filter through bugs by ID, Title, or Status instantly.
- **Secure Authentication**: Role-based access control to ensure users only see what they are supposed to.

## 🛠️ Tech Stack
- **Backend**: PHP 8.x
- **Database**: MySQL
- **Frontend**: HTML5, Modern CSS3 (Grid & Flexbox)
- **Icons/Fonts**: Google Fonts (Inter)
